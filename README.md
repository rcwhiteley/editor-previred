# Editor Previred

Editor web para archivos CSV en formato **Largo Variable por Separador** de Previred.

## Características

- Lectura de archivos CSV separados por `;` con encoding ISO-8859-1
- Vista tipo Excel con scroll virtual (soporta miles de filas)
- Campos codificados como selectores desplegables (AFP, Isapre, CCAF, etc.)
- Buscador con filtro en tiempo real y resaltado de coincidencias
- Indicador visual de celdas modificadas con conteo
- Exportación preservando formato y encoding original
- Archivo único `index.html` sin dependencias

## Uso

1. Abrir `index.html` en un navegador
2. Click en **Abrir CSV** y seleccionar el archivo
3. Editar los campos necesarios
4. Click en **Exportar CSV** para descargar

## Deploy

Compatible con GitHub Pages — simplemente habilitar Pages desde Settings apuntando a la rama principal.
